## Installation

Repository:
```json
    "repositories": {
        "backstage/crm": {
            "type": "vcs",
            "url": "git@github.com:arduinomaster22/crm.git"
        }
    }
```

You can install the package via composer:

```bash
composer require backstage/crm
```

You can publish and run the migrations with:

```bash
php artisan crm:install
```
